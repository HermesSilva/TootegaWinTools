# Assets

This folder contains images and media used in the README documentation.

## Required Files

- `logo.png` — Project logo (128x128 PNG with transparency)
- `screenshot.png` — Screenshot showing the extension in action

## Creating the Logo

The logo should represent a 7-Zip archive with Windows Explorer integration.
Suggested design: A folder icon merged with the "7z" text or archive symbol.

## Screenshot Guidelines

Take a screenshot showing:
1. Windows Explorer with a .7z file
2. The file opened as a virtual folder
3. Preview pane showing file contents
4. Context menu with extraction options
